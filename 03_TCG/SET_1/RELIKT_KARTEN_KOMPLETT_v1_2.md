# WAS SIND RELIKT-KARTEN?

**Relikt-Karten** sind permanente Ausruestung oder Feldeffekte!

**REGELN:**
- Max 3 Relikte gleichzeitig pro Spieler
- Kosten Energie zum Spielen
- Bleiben im Spiel bis zerstoert
- Zwei Typen:
  - **ATTACHED:** An einen Resonanten angelegt
  - **FIELD:** Globaler Effekt

**ATTACHED RELIKTE:**
- Wenn Resonant KO'd wird -> Relikt geht zum Discard
- Kann nur 1 Relikt pro Resonant

**FIELD RELIKTE:**
- Bleiben unabhaengig von Resonanten
- Effekt gilt fuer alle eigenen Resonanten
